# Werkstatt - Quick Start Guide

## Home Screen (Gallery)
- Your canvases appear as cards in a 2-column grid
- **Tap a card** to open that canvas
- **Tap the + button** to create a new canvas

---

## Drawing

### Open the Tool Panel
- Tap the **brush button** (bottom-left, shows current color + brush name)
- The panel slides up with all your tools

### Choose a Brush (8 types)
| Brush | Best For |
|-------|----------|
| Pen | Clean lines |
| Fine | Details, writing |
| Ball | Notes, sketching |
| Pencil | Shading, texture |
| Marker | Bold highlights |
| Water | Soft washes |
| Ink | Calligraphy |
| Brush | Painting |

### Adjust Size
- Use the SIZE slider (1-50px)

### Pick a Color
- Spin the **color wheel** for hue
- Adjust **saturation** and **brightness** sliders
- Or tap a **preset palette**: Bold, Pastel, Earth, Neon, Skin, Vintage
- Recent colors appear below the wheel

### Dismiss Panel
- Swipe down or tap outside

---

## Working with Images

### Import an Image
- Tap the **image icon** (bottom-right toolbar)
- Select from your gallery

### Edit an Image
- **Double-tap** the image to enter edit mode
- **Drag** to move it
- **Drag the corner handle** to resize
- **Tap the red trash button** to delete
- **Tap outside** the image to exit edit mode

### Draw Over Images
- Just draw! Your strokes appear on top of images

---

## Other Tools (Bottom Bar)

| Icon | Action |
|------|--------|
| Undo | Undo last stroke |
| Grid | Cycle grid (lines → dots → off) |
| Eraser | Toggle eraser mode |
| Image | Import image |

---

## Saving
- Your work **auto-saves** every 2 minutes
- Tap the **back arrow** to save and return to gallery
- The small spinner in the top bar shows when saving

---

Enjoy creating!
