A hyper-functional, infinite-canvas bullet journal for Android with Bauhaus-inspired aesthetics.

## Features

- **Infinite Canvas** - Pan and zoom endlessly
- **Bauhaus Grid** - Snap-to-grid for geometric precision
- **Ballpoint Drawing** - Smooth stroke capture
- **Auto-Save** - Never lose your work
- **Brutalist UI** - Massive blocky buttons

## Build

Push to GitHub and the APK will be automatically built via GitHub Actions.

Or build locally:

```bash
./gradlew assembleDebug
```

## Tech Stack

- Kotlin
- Jetpack Compose
- Room Database
- Material 3
